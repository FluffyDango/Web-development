Don't know what config files are?

Config files are like your cs:go console cheat sheet. Whatever you type in the config file will be pasted to the console. When you launch these files in the console, whatever was written to the config file will be pasted to the console.

This is useful if you want to save custom settings that when you launch the game it applies immediately.

They are written as "name".cfg and you can find them in:

C:\Program Files (86x)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg
