Don't know what config files are?

Config files are where you store your command data. All the commands you type in the config file will be pasted to the console in game. It's a quick way to launch your favorite settings or binds withouth much work.

This is useful if you want to save custom settings that you want to launch when opening Counter Strike.

They are written as [FileName].cfg and you can find them in:

C:\Program Files (86x)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg

Creating a cfg file

  1. Find the directory of cfg files from the link above.
  2. After navigating to the directory, create a random.txt text document and put the commands in there. You can copy paste this jumpthrow bind in here or add your own.
    3. alias "+jumpthrow" "+jump;-attack"; alias "-jumpthrow" "-jump"; bind x "+jumpthrow";
    Copy to clipboard
  3. Press save as and make the extension .cfg

Launching cfg files on startup

You can add any cfg file to autoload when starting CS:GO by editing the launcher options.
You can find them in steam by right clicking on CS:GO and pressing properties. Then you will see called Launch Options. All that's left is to add your commands

Launch options commands

  • +exec [filename].cfg

    • Will execute your created cfg file.

  • -nojoy

    • Will remove joystick support from the game, which may free up RAM.

  • -novid

    • Skips the intro video

That's it! Now every time you start CS:GO these specific commands will be launched.
