Don't know what config files are?

Config files are like your cs:go console cheat sheet. Whatever you type in the config file will be pasted to the console. When you launch these files in the console, whatever was written to the config file will be pasted to the console.

This is useful if you want to save custom settings that when you launch the game it applies immediately.

They are written as "name".cfg and you can find them in:

C:\Program Files (86x)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg

Launcher options

Creating a cfg file

  1. You can find the directory of cfg files in the text above.
  2. After navigating to the directory, create a random.txt text document and put the jumpthrow bind in there.
    3. alias "+jumpthrow" "+jump;-attack"; alias "-jumpthrow" "-jump"; bind x "+jumpthrow";
    Copy to clipboard
  3. Press save as and make the extension .cfg

Launching cfg files on startup

You can add any cfg file to autoload when starting CS:GO by editing the launcher options

  1. Right click on CS:GO and press properties
  2. Enter in launch options "+exec [filename].cfg"

That's it! Now every time you start CS:GO these specific commands will launch in console.

Additional options

Our recommendated:

  • -nojoy // will remove joystick support from the game, which may free up RAM.
  • -novid // skips the intro video
